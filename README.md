File.main java program ini menciptakan sebuah Kerajaan Pohon Bunga, yaitu Binary Search Tree (BST), yang diatur ketat berdasarkan Nilai Angka setiap bunga. Kepala Suku Tree adalah pengatur lalu lintas utama, yang menggunakan fungsi insert dan Agen Rahasia (insertRec) untuk memastikan bunga baru ditempatkan secara disiplin: kiri untuk nilai kecil, dan kanan untuk nilai besar. Setelah semua tertata, kita bisa meminta Parade Bunga (tampilIn/inorder) yang menampilkan semua data terurut dari terkecil hingga terbesar. Untuk mencari Si Paling Kaya (nilaiMax), kita tinggal lari terus ke ujung paling kanan pohon, sementara Polisi Sensor (cetakGenap) bertugas menyaring dan hanya menampilkan bunga-bunga yang Nilai Angkanya Genap. Terakhir, jika Anda mencari bunga kesukaan berdasarkan Nama, Detektif Khusus (cari/searchByName) harus menggeledah seluruh pohon secara rekursif, karena pencarian nama tidak mengikuti aturan urutan angka BST. Secara keseluruhan, file Main.java adalah Pusat Komando yang menjalankan semua fungsi hebat dari Tree.java dan Node.java tersebut.
